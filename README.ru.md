Никаких вкладок браузера. Никакого копирования в переводчик и обратно. Никаких ограничений по приложениям. Печатайте сообщение где угодно — в Slack, почте, чате игры, любой форме — нажмите свою горячую клавишу (по умолчанию **Ctrl+Shift+Z**, полностью настраивается), и перевод появится прямо в том же поле. Буфер обмена останется нетронутым.

## Чем это лучше

- **Работает везде** — в любом активном текстовом поле, в любом приложении, без ограничений.
- **Никогда не подводит** — пять AI-провайдеров (Gemini, OpenAI, Anthropic, DeepSeek, OpenRouter) плюс локальные модели (Ollama, LM Studio) с автоматическим переключением между выбранными моделями, если одна недоступна, тормозит или заблокирована.
- **Передаёт то, что вы имели в виду** — режим перевода «Expressive» переводит тон, сленг, эмодзи и ненормативную лексику как есть, вместо привычного корпоративного смягчения от AI-провайдера; можно переключиться на «Standard», если нужен более консервативный вариант по умолчанию.
- **Просто по умолчанию, гибко при желании** — язык перевода (с избранными языками для быстрого переключения), горячая клавиша, AI-провайдер и порядок переключения между моделями — всё в одной панели настроек, плюс две встроенные темы (Calm Dark, Cyberpunk).
- **Живёт в трее** — сворачивается в фон, возвращается одним кликом; опция «Запуск с Windows» сразу открывается в трее.

## Быстрый старт

```
pip install -r requirements.txt
python src/main_app.py
```

Готовые установщики: `python build_exe.py` (Windows) или `python3 build_linux.py` (Linux).

## Поддерживаемые платформы

Windows (нативная глобальная горячая клавиша) и Linux/X11 (слушатель по аппаратным кодам клавиш). Сессии только на Wayland пока не поддерживаются — используйте XWayland.

## Список изменений

### 1.1.0
- **Добавлен OpenRouter в качестве пятого AI-провайдера.** OpenRouter даёт доступ к сотням моделей от сторонних хостеров, включая доработки сообщества (по умолчанию используется линейка "Dolphin" от Cognitive Computations), которые не применяют те же отказы по контент-политике, что и основные провайдеры — полезно, когда Gemini/OpenAI/Anthropic/DeepSeek отказываются переводить в остальном законный, но грубый или откровенный текст.
- **Исправлена чрезмерная блокировка в режиме Expressive.** Пороги безопасности Gemini `HARASSMENT` и `SEXUALLY_EXPLICIT` теперь полностью ослаблены (`BLOCK_NONE`) вместо частичного ослабления (`BLOCK_ONLY_HIGH`), поэтому обычный грубый или откровенный, но не связанный с ненавистью текст больше не блокируется молча. Пороги для разжигания ненависти и опасного контента не изменены.
